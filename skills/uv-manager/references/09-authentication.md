# Authentication

## `uv auth login` - Login to service

```bash
uv auth login pypi.org
uv auth login https://my-index.example.com
```

---

## `uv auth logout` - Logout

```bash
uv auth logout pypi.org
```

---

## `uv auth token` - Show token

```bash
uv auth token pypi.org
```

---

## `uv auth dir` - Show credentials directory

```bash
uv auth dir
```
