# Cache Management

## `uv cache clean` - Clear cache

```bash
uv cache clean                   # Clear all cache
uv cache clean requests          # Clear specific package
```

---

## `uv cache prune` - Prune unreachable objects

```bash
uv cache prune
```

---

## `uv cache dir` - Show cache directory

```bash
uv cache dir
```

---

## `uv cache size` - Show cache size

```bash
uv cache size
```
